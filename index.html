Provérbios de Go

Simples, Poético, Conciso

Não comunique por compartilhamento de memória, compartilhe memória por comunicação.

Concorrência não é paralelismo.

Canais orquestram; semáforos serializam.

Quanto maior a interface, mais fraca a abstração.

Faça o valor zero útil.

interface{} diz nada.

O estilo de Gofmt não é favorito de ninguém, ainda gofmt é o favorito de todos.

Uma pequena cópia é melhor que uma pequena dependência.

Syscall deve estar sempre resguardada com tags de build.

Cgo must always be guarded with build tags.

Cgo não é Go.

Com a unsafe package não existem garantias.

Limpo é melhor que esperto.

Reflection nunca é claro.

Errors são valores.

Don't just check errors, handle them gracefully.

Desenhe a arquitetura, nomeie os componentes, documente os detalhes.

Documentação é para os usuários.

Não entre em pânico.

Menções honrosas

Aceite Interfaces, retorne tipos concretos.
